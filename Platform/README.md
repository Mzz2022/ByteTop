# ByteTop-Platform

ByteTop埋点项目数据平台

## 项目描述

## 技术栈

- **React v18**: 开发框架
- **TypeScript**: 开发语言
- **Vite**: 构建工具
- **HeroUI**: UI组件库
- **React Router**: 路由管理
- **Tailwind CSS**: CSS 框架
- **reactbits**: 动画效果
- **Prettier-ESLint**: 代码质量规范

## 目录结构

```
platform/
├── public/
├── src/
│ ├── assets/# 静态资源
│ ├── components/ # 共享组件
│ ├── pages/ # 页面组件
│ ├── store/ # 状态管理
│ ├── utils/ # 工具函数
│ ├── App.tsx # 应用主组件
│ ├── main.tsx # 应用入口
│ └── ... # 其他文件
├── .eslintrc.js # ESLint 配置
├── tailwind.config.js # Tailwind CSS 配置
├── tsconfig.json # TypeScript 配置
├── package.json # 项目依赖和脚本
└── README.md # 项目说明
```

## 安装

pnpm install （统一使用pnpm！！！）

## 开发

pnpm run dev

## 构建

pnpm run build
